#  Basic Single Page Application (SPA) using React & Vite

This project is developed as part of **Experiment 1** for the course **Full Stack – II (23CSH-382)**.  
The aim of this experiment is to create a **basic Single Page Application (SPA)** using a modern frontend framework and understand core SPA concepts.

##  Experiment Details
- **Experiment No.:** 1  
- **Title:** Create a Basic SPA Using Modern Frontend Framework  
- **Framework:** React (Vite)

##  Tech Stack
- React  
- Vite  
- JavaScript (ES6+)  
- React Router DOM  
- HTML & CSS  
- Node.js & npm  
- Git & GitHub  

##  Steps to Run the Project
1. Clone the repository  
`git clone <your-github-repository-url>`

2. Navigate to the project folder  
`cd my-spa`

3. Install dependencies  
`npm install`

4. Start the development server  
`npm run dev`

5. Open in browser  
`http://localhost:5173` 

##  What We Learned
- Working of Single Page Applications  
- Component-based architecture in React  
- State management using `useState` and `useEffect`  
- Client-side routing without page reload  
- API integration in frontend applications  
- Version control using Git and GitHub  

##  Learning Outcomes
- Ability to design a basic SPA  
- Understanding of modern frontend development workflow  
- Hands-on experience with routing and state management  
- Knowledge of deploying frontend applications
